# 💸 Expense Tracker (CLI + Streamlit)

## 🚀 Project Overview

A simple yet powerful **Expense Tracker** that allows users to:

- Add expenses 💸
- View all expenses 📋
- Filter by category 📂
- View total spending 📊
- Category-wise summary 📈
- Persistent storage using JSON 💾

Built in two versions:
- 🖥️ CLI (Command Line Interface)
- 🌐 Streamlit Web App

---

## 🧠 Key Learnings

- Working with **JSON for data storage**
- Structuring data using **lists & dictionaries**
- Building **modular functions**
- Handling edge cases (empty file, invalid input)
- Creating **interactive UI using Streamlit**
- Thinking like a developer (data flow, persistence)
